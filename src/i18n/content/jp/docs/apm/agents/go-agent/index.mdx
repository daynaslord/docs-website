<LandingPageHero>
  <HeroContent>
    New RelicのGoエージェントは、Go言語アプリケーションとマイクロサービスを監視して、以下を支援します。

    * パフォーマンス問題の特定と解決。

    * 顧客体験の向上とデータに基づくビジネス上の意思決定。

    * [APM](/docs/apm/new-relic-apm/getting-started/introduction-new-relic-apm)、[インフラストラクチャ](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure)、その他のNew Relic製品の詳細なダッシュボードからのデータを監視。

    * [ダッシュボード](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/)からデータのクエリ、ビジュアライゼーション、共有を行います。

      エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Goエージェントのリリースノート](/docs/release-notes/agent-release-notes/go-release-notes)を参照ください。

      </ButtonLink>
    </ButtonGroup>

    クレジットカードは不要です。すでにアカウントをお持ちの場合[ログイン](http://one.newrelic.com/)。
  </HeroContent>
